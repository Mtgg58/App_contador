# App Contador

este es de una aplicación de contador desarrollada utilizando Flutter y el editor VS Code

## caracteristicas

**HomeScreen:** cuenta con tres botones de accion flotantes para sumar +1, restar -1 y reiniciar el valor a cero

**InfoScreen:** pantalla secundaria con los datos de identificacion del estudiante.

**Navegacion:** uso de rutas nombradas (`Map<String, WidgetBuilder>`) para transicionar limpiamente entre pantallas
